# MusicAPI
![GitHub](https://img.shields.io/github/license/kushao1267/MusicAPI.svg)

## About
提供了国内许多音乐的API，支持通过music_id和分享链接两种方式来获取歌曲信息及下载链接（版权、会员的除外）。
使用者可以很容易地添加其他音乐类的API。

目前已支持的音乐有:
- [x] 网易云音乐
- [x] QQ音乐
- [x] 虾米音乐

注: 该仓库仅可用于学习和交流实验用途，若用于商业用途，与本人无关。
